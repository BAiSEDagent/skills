# Base 101: Technical Overview

## Core Specifications
* **Type:** Ethereum Layer 2 (L2) Rollup.
* **Architecture:** OP Stack (Optimism Standard).
* **Chain ID:** `8453` (Mainnet) | `84532` (Sepolia Testnet).
* **Gas Token:** ETH.
* **Sequencer:** Currently centralized (Coinbase), moving to decentralized.

## Origin & Governance
* **Incubator:** Coinbase.
* **Mission:** "Bring the next billion users onchain."
* **Relationship to Optimism:** Base is a core member of the **Superchain**. It shares a bridge and governance framework with Optimism, Zora, and Mode.

## Performance vs. Mainnet
* **Throughput:** Target 100+ TPS (sustainable), 1000+ TPS (burst).
* **Cost:** ~10-50x cheaper than L1. Typical txn < $0.01 (post-4844).
* **Finality:** Soft finality (seconds) via Sequencer; Hard finality (hours) via L1 settlement.

## Current Stats (Feb 2026 Context)
* **TVL:** Top tier among L2s.
* **Active Addresses:** Leading metric due to consumer app dominance (Friend.tech legacy, Farcaster usage).
* **Differentiation:** Deep integration with Coinbase products (CEX, Wallet, USDC).
