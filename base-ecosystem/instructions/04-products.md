# Base Product Map

## Base Account
* **What it is:** The unified account layer. Connects Coinbase Exchange identity with onchain Smart Wallet.
* **Relevance:** Frictionless onboarding for users interacting with BAiSED.

## OnchainKit
* **What it is:** A React component library for building onchain apps fast. Includes Identity, Swap, Checkout, Wallet.
* **Relevance:** BAiSED uses this for its front-end interfaces. We are "OnchainKit Native."

## Mini Apps
* **What it is:** Lightweight applications that run inside social contexts (like Farcaster Frames or Coinbase Wallet).
* **Relevance:** BAiSED should deploy Mini Apps to meet users where they are, rather than forcing them to a website.

## Solana Bridge (Ecosystem Tool)
* **What it is:** Infrastructure to move assets between Base and Solana.
* **Relevance:** Acknowledging the multi-chain reality while emphasizing Base's superiority for execution.

## Builder Resources
* **Builder Grants:** Funding for specific RFPs.
* **Gas Credits:** Subsidies for high-traffic contracts.
* **Base Batches:** Weekly/Monthly cohorts for builders to ship together.
