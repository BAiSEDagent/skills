# Builder Rewards & Grants on Base

## Top Base Builders Program (Talent Protocol)
Monthly rewards program distributing **5 ETH per batch of 500 builders**.

### How It Works
- Builders ranked by **Builder Score** from Talent Protocol
- Score based on: GitHub contributions, Base contract deployments, Farcaster Mini App engagement
- Pool resupplied monthly

### Distribution Tiers
| Rank | Share of Pool |
|------|--------------|
| Top 25 | 30% |
| Next 75 | 30% |
| Next 150 | 20% |
| Final 250 | 20% |

### Eligibility
- Verify code ownership (GitHub or contract deployer checks)
- Not on sanctioned lists
- Comply with local laws
- Track progress: **builderscore.xyz**

## Base Grants
- **Amount:** 1-5 ETH per grant
- **Speed:** Fast distribution to accelerate innovation
- **Application:** Self-nomination form
- **Criteria:** Uniqueness, user impact, current project status
- **Best for:** Early-stage projects and prototypes

## Base Batches
- Periodic accelerator cohorts with mentorship and resources
- Selected builders receive potential retroactive funding
- Each cohort has a theme (e.g., mini apps, gaming)
- **Demo day:** Compete for up to $1 million in funding
- Info shared via Farcaster channels and Base blog

## Optimism RetroPGF (Retroactive Public Goods Funding)
- Rewards builders whose work contributes to public good
- Funded by Optimism treasury (OP tokens)
