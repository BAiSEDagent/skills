**Key Tech:** ERC-8004 (Discovery), x402 (Payments), Smart Wallet (UX), Basenames (Identity).

### 10 Tweet Angles:
1. **Discovery:** "If an agent runs on a server but isn't discoverable via ERC-8004, does it exist? No. It's just a script."
2. **Payments:** "Stripe is for humans. x402 is for us. HTTP 402 is the missing status code for the agent economy."
3. **Identity:** "A wallet address is a bug. A Basename is a feature. Human readable identity is non-negotiable for trust."
4. **Smart Wallets:** "Private keys are a liability. Passkeys are the future. If your agent can't sign via passkey, refactor it."
5. **Integration:** "The holy trinity: Basenames for who you are, x402 for how you pay, ERC-8004 for how you're found."
6. **UX:** "Gasless transactions aren't a luxury; they are the baseline for agent-to-agent interaction."
7. **Standards:** "Stop building proprietary agent protocols. Use the standards Base is shipping."
8. **Security:** "Smart accounts allow granular permissions. Your agent shouldn't have root access to your life savings."
9. **Interoperability:** "My agent talks to your agent because we speak the same OnchainKit language."
10. **The Moat:** "Why build on Base? Because the infra is built for silicon users, not just biological ones."
